Exploration 3.3.17.

Assume \(f:V \rightarrow W\) is a linear map. Let \(\{\vec{v_1},\vec{v_2},\ldots,\vec{v_n}\}\) be a set of vectors in \(V\text{,}\) and set \(\vec{w_i} = f(\vec{v_i})\) \begin{itemize} \item If the set \(\{\vec{w_1},\vec{w_2},\ldots,\vec{w_n}\}\) is linearly independent, must the set \(\{\vec{v_1},\vec{v_2},\ldots,\vec{v_n}\}\) also be linearly independent? \item If the set \(\{\vec{v_1},\vec{v_2},\ldots,\vec{v_n}\}\) is linearly independent, must the set \(\{\vec{w_1},\vec{w_2},\ldots,\vec{w_n}\}\) also be linearly independent? \item If the set \(\{\vec{w_1},\vec{w_2},\ldots,\vec{w_n}\}\) spans $W$, must the set \(\{\vec{v_1},\vec{v_2},\ldots,\vec{v_n}\}$ also span $V\text{?}\) \item If the set \(\{\vec{v_1},\vec{v_2},\ldots,\vec{v_n}\}\) spans $V$, must the set \(\{\vec{w_1},\vec{w_2},\ldots,\vec{w_n}\}$ also span $W\text{?}\) \item In light of this, is the image of the basis of a vector space always a basis for the codomain?
