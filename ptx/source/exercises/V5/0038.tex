
\begin{exerciseStatement}

\end{array}\right] \right\} \)is linearly dependent.
\end{minipage}\end{center}
    


\begin{enumerate}[(a)]
\item  Write an equivalent statement using a vector equation.
\item  Explain why your statement is true or false.
\end{enumerate}
