Let \(A\) be a \(4 \times 4\) matrix with determinant \(-7\text{.}\) \​begin{enumerate}[(a)] \item Let \(B\) be the matrix obtained from \(A\) by applying the row operation \(R_3 \to R_3+3R_4\text{.}\) What is \(\det(B)\text{?}\) \item Let \(C\) be the matrix obtained from \(A\) by applying the row operation \(R_2 \to -3R_2\text{.}\) What is \(\det(C)\text{?}\) \item Let \(D\) be the matrix obtained from \(A\) by applying the row operation \(R_3 \leftrightarrow R_4\text{.}\) What is \(\det(D)\text{?}\) \end{enumerate} \end{problem} \​begin{solution} \​begin{enumerate}[(a)] \item Adding a multiple of one row to another row does not change the determinant, so \(\det(B)=\det(A)=-7\text{.}\) \item Scaling a row scales the determinant by the same factor, so so \(\det(B)=-3\det(A)=-3(-7)=21\text{.}\) \item Swaping rows changes the sign of the determinant, so \(\det(B)=-\det(A)=7\text{.}\) \end{enumerate}
